                                                                  Линукс команды: 

1.  pwd (Print Working Directory) -> Возвращает абсолютный путь до на шей папки.
2.  ls (List) -> Вызывает весь список файлов и папок находящийся там(так же можно указать путь где вызывать эту команду)
3.  cd (change diectory) <dir_name> -> Проводник до определенной папки 
      cd .. -> Перемещаемся на уровень ниже (назад) 
      cd /home/<user_name>/<dir_name> -> Перемещение по абсолютному пути 
      cd -> Домашняя директория
4.  mkdir <name_dir> -> создать директорию
5.  rmdir <name_dir> -> Удалить директорию, если она пустая
6.  clear (ctrl+L) -> Очистить терминал, при нажатии гор.клавиши, спускает вниз.
7.  rm -rf <name_dir> -> Принудительное удаление директории
8.  touch <file.name> -> Создать файл
9.  rm <file.name> -> Удаление файла 
10. cat <file.name> -> Посмотреть содержимое файла
11. nano <file.name> -> Открывает файл(если существует), создает(если его нет) 
      1. ctrl + X - Что бы сохранить 
      2. ctrl + Y - Что бы подтвердить
      3. Enter - закрыть меню
12. ls -a -> выводит список всех файлов(включая скрытые файлы) 
      ls -la -> выводит весь список файлов, только с подробной информацией
13. mv <file.name> <dir_name> -> Перемещает выбранный файл в выбранную папку. 
      Так же служит для переименования.
14. cp <file.name> <dir_name> -> Копирует файл в выбранную папку. 
15. echo <text> > <file.name> -> Задает содержимое файлу
16. history -> Открывает историю всех введенных команд за все время.
17. find <name> -> Ищет файл или переходит 
    find /home -name <file.name> - Ищет везде схожие файлы
18. sudo <command>  -> Выполняет команду, с полным доступом.
